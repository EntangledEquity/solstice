## Welcome to mini-project SOLSTICE!

A dive into understanding Mathematical Theory of Bayesian Statistics and Singular Learning Theory.
Along the way we will explore optimization methods like Monte carlo Markov Chains, SGLD for sampling.

The end goal is to understand and implement methods like the Local learning coefficient on Neural Networks, a measure for studying model complexity of Singular models with a rigorous understanding of aforementioned topics.

This repository contains tasks for Weeks 0–3, designed by Project Leads Smitali and Bhavi. The tasks are intended to help readers better understand the referenced material and gradually develop their own theoretical understanding and proofs.

### Content
- Task 0
  
  Basic probability theory. Intution behind Bayesian framework and a toy example with dataset of      size 10, to find the underlying distribution. Food for thought question.
- Task 1
  
  Bayesian framework for conditionally independent variables. Comparing Bayesian optimization to      point wise methods like Gradient Descent. Losses, errors and free energy. Estimation methods
- Task 2 Part A
  
  Monte Carlo Markov chain methods - Why? How? Limitations.
  Metropolis method, Gibbs Sampler, Hamiltonian Monte Carlo
- Task 2 Part B
  
  Implementation of MCMC methods for QR code correction and Bot v/s Human classification for Social    Networks
- Task 2 Part C
    
  A walk through the Distilling Singular Learning Theory 1 blog. Building up from Fisher              Information  Matrix with proof (except Asymptotics, those assumed) to Bayesian information          criterion and to singular models, RLCT.
