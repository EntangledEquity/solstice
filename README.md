## Welcome to mini-project SOLSTICE!

A dive into understanding Mathematical Theory of Bayesian Statistics and Singular Learning Thoery.
Along the way we will explore optimization methods like Monte carlo Markov Chains, SGLD for sampling.
The end goal is to understand and implement methods like the Local learning coefficient on Neural Networks, a measure for studying model complexity of Singular models.

This repository currently contains task 0 and week 1 task designed by Project Leads Smitali and Bhavi.

The aim of the project is to develop a rigorous understanding of the aforementioned topics.

Resources used so far:
Mathematical theory of Bayesian Statistics by Sumio Watanabe
